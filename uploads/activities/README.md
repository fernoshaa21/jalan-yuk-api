# Activity Images (Portfolio Seed)

Folder ini dipakai untuk image activity lokal yang dilayani sebagai static files oleh backend NestJS.

## Naming convention yang direkomendasikan

Gunakan pola konsisten:

- `<category>-<index>.jpg`

Contoh:

- `adventure-1.jpg`
- `nature-2.jpg`
- `city-tour-3.jpg`
- `water-sport-4.jpg`

Category yang dipakai backend:

- `adventure`
- `nature`
- `culture`
- `culinary`
- `city-tour`
- `water-sport`
- `family`

## Public URL

Jika backend berjalan di `http://localhost:3000`, file bisa diakses lewat:

- `http://localhost:3000/uploads/activities/adventure-1.jpg`

## Catatan

- File yang ada saat ini adalah placeholder untuk data awal portfolio.
- Nantinya bisa diganti file real saat web admin upload sudah tersedia.
